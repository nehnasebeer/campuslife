Campus Life Super App
Project Description & Purpose
The Campus Life Super App is a multi-page, responsive web application designed to help students easily access everyday campus essentials. The site provides quick access to events, dining hall hours, campus information, and interactive tools — all organized in one easy-to-use platform.
The purpose of this project is to create a centralized, student-friendly experience using clean UI, responsive layouts, and interactive JavaScript features.

Technologies Used
HTML5 – structure & semantic layout
CSS3 – custom styling & responsive design
Bootstrap 5 – grid system & prebuilt components
JavaScript (ES6) – interactivity & event-driven programming
External API – integrated for dynamic content (Giphy API)
GitHub Pages – deployment
Git & GitHub – version control & project management

Setup & Deployment Instructions
How to Run the Project Locally
Clone the repository:
git clone https://github.com/nehnasebeer/campuslife
Open the folder in VS Code.
Right-click index.html → “Open With Live Server”
(or simply open the file in browser).

Deployed Website
The live version is available at:
